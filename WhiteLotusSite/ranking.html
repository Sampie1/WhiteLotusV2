Ranks


Hier vindt je alle ranks van White Lotus V2

Founder

Founder is de hoogste rank van White Lotus V2. Deze rank houd in dat je eigenaar bent
van de server.

Co-Founder

Co-Founder is de een-na-hoogste rank van
White Lotus V2. Leden met deze rank beheren
de server zodat alles goed blijft lopen.

Management

Management regelt alles omrent de mail. ook
regelt het Management events en giveaways
(uiteraard niet op eigen kosten).

Support-Team

SupportTeam zorgt er voor dat de server netjes
blijft daarnaast helpt het SupportTeam andere server leden die een ticket hebben aangemaakt.

Development

Development rank krijg je door het maken van eigen discordbots voor onze server. Deze rank houd in dat je discordbots op aanvraag kan maken.

Server Booster

Server Booster deze rank krijg je op het moment dat je onze server minimaal 1x een server boost hebt gegeven.

Vrienden

De vrienden rol krijg je als je bevriend bent met
één van de Founders/Co-Founders. Deze rol
houd in: niet veel actief maar wel bevriend.

Member

De member rol krijgt iedereen door het joinen van de server. Deze rol houd in dat je nieuw bent
of niet zo actief.
